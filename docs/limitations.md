# Limitations

CellFateBench is a research and portfolio benchmark project.

## Current Limitations

- The benchmark begins with controlled synthetic datasets rather than large experimental atlases.
- Synthetic data is useful for known hidden truth, but it cannot capture the full complexity of real biological systems.
- Initial tasks are designed for reasoning evaluation, not biological discovery.
- The project does not make clinical or diagnostic claims.
- The project does not replace expert single-cell analysis or experimental validation.

## Why These Boundaries Matter

Benchmark design requires controlled answer keys. This project therefore prioritises evaluability, transparency, and reproducibility over biological scale in the first version.

Future versions can add public real-world single-cell datasets after the benchmark framework is established.
