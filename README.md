# CellFateBench: A Single-Cell Benchmark for Trajectory, Spatial, and Topological Analysis

A benchmark-design project for evaluating advanced reasoning across single-cell genomics tasks involving trajectory structure, spatial expression patterns, and topological persistence.

This repository is under active development.

## Scope

This is a research, learning, and portfolio project. It is designed to demonstrate scientific software engineering, single-cell genomics reasoning, benchmark task design, oracle outputs, validators, calibration-style review, reproducibility, Docker, CI, and evidence-backed documentation.

It is not intended for clinical diagnosis, regulated biomedical reporting, or production laboratory use.

## Planned Benchmark Task Families

1. Trajectory and pseudotime reasoning
2. Spatial gene-pattern reasoning
3. Topological persistence reasoning

## Target Capability

The project is designed to demonstrate the ability to build original computational benchmark tasks grounded in single-cell genomics workflows, with public task prompts, hidden answer keys, oracle outputs, deterministic validators, scoring reports, calibration logs, and reproducible execution.

## Development Status

Project foundation has been initialised. Dataset generation, task construction, validators, scoring, reports, Docker, CI, and full documentation will be added in subsequent phases.
